# SharedMQ:进程间消息队列

> 是一个进程间通信组件：基于IO事件、基于共享内存实现的消息队列

### 特性
- 支持单消费者进程与单生产者进程
- 支持多消费者进程与多生产者进程
- 支持单消费者线程与单生产者线程（但是不如直接用内存数据结构，不推荐使用）
- 不支持多生产者线程与多消费者线程
